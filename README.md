# 🕹️ Mini Arcade — Two Browser Games Built with Claude Fable 5

**▶ Play here: https://pranavan04.github.io/arcade-games/**

Two single-file HTML5 games generated with [Claude Fable 5](https://www.anthropic.com) (Anthropic) as a weekend experiment in AI-assisted rapid prototyping. No game engine, no dependencies, no build step — each game is one self-contained HTML file.

## 🏎️ Turbo Rush — Night Highway Racer
Dodge traffic across four lanes, grab turbo stars, and survive as far as you can while the speed keeps climbing.

**Controls:** ← → / A D to steer · hold ↑ / W for turbo boost · P to pause

## 🚀 Nova Surge — Arena Survival Shooter
Fight endless waves of four enemy types, collect power cores (health, rapid fire, spread shot, shield) and dash through danger with invincibility frames.

**Controls:** WASD / arrows to move · mouse to aim · click or Space to fire · Shift to dash · P to pause

## Tech
- Vanilla JavaScript + HTML5 Canvas, ~600 lines per game
- Delta-time game loop, collision detection, particle effects, difficulty scaling
- Runs in any modern browser — just open the HTML file or visit the link above
